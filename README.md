# Java Sign Up & Sign In System (Console Project)

This is a beginner-level Java console application built to practice **Object-Oriented Programming (OOP)** concepts and basic program flow.

The project simulates a simple **Sign Up and Sign In system**, similar to how real applications manage user accounts, but in a simplified form for learning purposes.
